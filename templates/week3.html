The Deep Dive

  • A Practical Guide to Building Custom Directives

    We only used the built-in directives that Angular provides for us. But one of Angulars central (and most powerful) features is the ability to define your own directives. Essentially, you can create your own HTML elements that have specific javascript/angular code attached to them. Creating directives lets you create custom UI elements that you can repeat all over your site.

  • Services in AngularJS, Simplified

    Like with directives, we only used the built-in services provided for us by Angular. Writing your own services allows you to share application logic and code across controllers, as well as to store data for the lifetime of an application. The true meat of an application should lie in its services, instead of being written directly into controllers. This article breaks down some of the ideas behind services and walks through the process of creating one.

  • Promises: the definitive guide

    We worked with promises a lot. But they're way more powerful and useful than we had time to dig into. This guide will show you all the cool things promises can do that you didn't know about, and teaches you the ways that using prommises effectively will help structure an application effectively.
