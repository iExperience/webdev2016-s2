More Power, Less Pain

JavaScript is not easy. It's hard to work with, annoying at times, and super picky about every single character you type. But JavaScript is what we have to work with, and JavaScript can do a lot. We hope that you feel like you can wield it to solve problems, not just type and fix syntax errors until something happens.

We've picked out the resources below to provide a starting point for some of the things you might be more likely to want to do with Javascript. There's an insanely huge world of frameworks, libraries, tools, and more out there. We only scratched the surface.

Animation and Interaction

  • Velocity.js

    Animation is everywhere on the web, but it's hard to do well. Among other issues that we can't really solve for you (like when should you use it? How long should that fade take?), one that we can help you avoid is performance. A lot of animations run slowly. VelocityJS is a library that does a great job of making animations run smoothly without you needing to geek out about why.

  • SwiperJS

    There are a thousand carousels and image sliders out there. SwiperJS is the most powerful and performant ones, while still being easy to use. It lets you configure almost anything you want, and makes it easy to create swipable slides and boxes on a page, that work well on mobile devices as well.

Useful Libaries

  • Lodash

    Lodash makes a lot of tasks involving working with objects and arrays way less painful. Need to apply a function to every element in an array? Need to convert an object to an array of only its keys or values? Need to filter an array of objects by one property? All these kinds of tasks are made trivial with lodash. Bonus: lodash's documentation is phenomenal.

  • D3

    D3 is a library that lets you take a lot of data and display it in a beautiful manner within a browser. You can create your own infographics without much pain&emdash;there are some truly impressive displays built with D3.

Coding with JS

  • Eloquent JavaScript

    A textbook that is available freely to read online, Eloquent JavaScript is an excellent resource for thinking about JavaScript as a programming language. It's written in a conversational style and is extremely readable (I'd almost hesitate to call it a textbook). If you want to become a master of writing JavaScript, this is a great place to start.
